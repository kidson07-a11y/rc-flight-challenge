# RC Flight Challenge

A fun, interactive Jeopardy-style game show about LiPO Batteries and Radio Control Systems!

## Game Categories

1. **LiPO Battery Basics** - Learn about Lithium Polymer batteries
2. **Radio Control Systems** - Understand transmitters, receivers, and ESCs
3. **Safety & Performance** - Explore voltage, C-ratings, and battery management
4. **Aircraft Components** - Discover motors, servos, and propellers

## How to Play

- Click on any point value (100-500)
- Read the question carefully
- Click "Show Answer" to reveal the correct answer
- The tile will be marked when answered
- Click "Reset Game" to start over
